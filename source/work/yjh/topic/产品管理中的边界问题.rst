产品管理中的边界问题
===================
产品规划和SE角色在产品管理中的边界问题，同时映射产品中心和研发中心在产品管理中的边界问题。

这里的产品管理特指产品管理中的一个场景，是从用户场景到需求定义、规格制定、方案设计的过程。

```plantuml
@startuml
|产品中心|
start
:调研用户场景;
:编制产品需求包文档;
|#pink|控制点|
#pink:评审产品需求包文档;
|研发中心|
:制定产品设计规格;
|控制点|
#pink:评审产品设计规格;
|研发中心|
:设计产品方案;
|控制点|
#pink:评审产品设计方案;
stop
@enduml

```
